# Sourav Jangra – Developer Portfolio

This is my personal developer portfolio website.

It showcases:

- My frontend-focused projects
- Case studies explaining how I built them
- Notes on what I learned while working on real applications
- My resume and contact information

## Tech Stack

- HTML
- CSS
- JavaScript

## Sections

- Home – Introduction and current focus
- Projects – Selected work with live demos and GitHub links
- Notes – Learning notes written while building projects
- Resume – Web-based resume with downloadable PDF
- Contact – Ways to reach me

## Projects

Each project lives in its own repository and is deployed separately.
This portfolio acts as a central hub linking to:

- Live project demos
- Source code repositories

## Deployment

The portfolio is deployed using Vercel and automatically updates on every push to the main branch.
